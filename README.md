# magic-fight
##下载之后双击打开index.HTML即可开始游戏
##游戏内容概述
🔥 火系 (Fire) - 擅长持续灼烧与爆发伤害

- 火球术 (CD: 2s | 消耗: 10): 发射火球，附带灼烧效果
- 炎爆阵 (CD: 8s | 消耗: 30): 自身周围爆发，造成范围伤害并击退
- 烈焰冲刺 (CD: 6s | 消耗: 20): 向前突进一段距离，对路径上的敌人造成伤害
- 火墙术 (CD: 12s | 消耗: 35): 在面前生成一道火墙，穿越的敌人会被严重灼烧
- 陨石雨 (CD: 20s | 消耗: 50): 在敌人当前位置召唤巨大的陨石砸下，造成毁灭性范围伤害
💧 水系 (Water) - 擅长控制与减速限制

- 冰霜射线 (CD: 3s | 消耗: 15): 发射射线，附带减速效果
- 水牢术 (CD: 10s | 消耗: 25): 在敌人脚下召唤水牢，命中造成禁锢
- 水流喷射 (CD: 5s | 消耗: 20): 持续向前方喷射水流，造成多段伤害并击退
- 冰霜护甲 (CD: 15s | 消耗: 30): 为自己附加护甲，减少受到的伤害并使攻击者减速
- 暴风雪 (CD: 18s | 消耗: 45): 召唤大范围暴风雪，持续造成伤害并强力减速所有敌人
🪨 土系 (Earth) - 擅长防御护盾与地形限制

- 飞石连击 (CD: 4s | 消耗: 20): 发射三颗飞石，附带微弱减速
- 大地之盾 (CD: 12s | 消耗: 40): 召唤高额护盾，破裂时反伤周围敌人
- 地刺突袭 (CD: 8s | 消耗: 25): 从地下召唤一排地刺向前方蔓延，造成伤害和短暂击飞
- 泥沼术 (CD: 14s | 消耗: 35): 在目标区域生成泥沼，极大降低敌人移速并持续吸蓝
- 地震波 (CD: 18s | 消耗: 45): 以自身为中心引发大范围地震，造成伤害并眩晕（禁锢）所有敌人
🌪️ 风系 (Wind) - 擅长机动性与飞行物阻挡

- 风刃 (CD: 1.5s | 消耗: 8): 基础攻击，弹道极快，伤害较低
- 旋风斩 (CD: 6s | 消耗: 25): 发出缓慢向前的龙卷风，持续多段伤害并附带小幅击退
- 风墙 (CD: 12s | 消耗: 30): 在面前生成一道风墙，持续4秒，阻挡所有敌方飞行道具
- 飓风术 (CD: 15s | 消耗: 45): 在目标区域召唤强大飓风，短暂延迟后将敌人浮空并造成巨额伤害
- 顺风之息 (CD: 20s | 消耗: 20): 为自己附加极速状态，移动速度提升60%，持续5秒
